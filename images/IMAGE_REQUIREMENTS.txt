====================================
TIPS FOR PROJECT IMAGES:
====================================

- Use actual screenshots of your projects if available
- If you don't have screenshots, you can use:
  * Code editor screenshots with relevant code
  * API documentation screenshots
  * Architecture diagrams
  * Placeholder images from services like:
    - Unsplash.com (free stock photos)
    - Pexels.com (free stock photos)
    - Lorem Picsum (placeholder images)

- Optimize images before adding:
  * Use tools like TinyPNG.com or Squoosh.app
  * Target file size: under 200KB per image
  * Maintain aspect ratios to avoid distortion
